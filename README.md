## Open the app

`C:\Users\Mason\Documents\MLO\OPEN_MLO_LEARNING_APP.bat`

Or open:

## What it includes

- Five-day dashboard.
- Day-by-day syllabus execution plan.
- All 150 flashcards from the study deck.
- Flashcard status tracking: known, missed, unseen.
- Quiz mode by category.
- Math drills.
- Timing drills.
- Miss log.
- Links to all printable PDF manuals.
- Local progress storage in your browser.
- Progress export as a JSON file.

## How progress works

Progress is saved in the browser on this computer using local storage. It stays available when you close and reopen the app in the same browser.

Use `Export progress` if you want a backup file.

## How to move it to another computer

Copy the entire folder:

`C:\Users\Mason\Documents\MLO`

The app needs the `interactive_learning_app` folder and the printable PDF files to keep all links working.
